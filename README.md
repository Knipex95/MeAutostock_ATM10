# MeAutostock_ATM10
A version from the ME Autostock software for ATM10.

You need:
1. Advance PC
2. Advance Monitor
2. Me Bridge

Install to a advance PC:
```
pastebin get pMRU9wr4 gitget
gitget Knipex95 MeAutostock_ATM10 main
```
After that make a restart.

Add Item to the Autocraft:
```
Stockpile [Amount] [Item]
```
